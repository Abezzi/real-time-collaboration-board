# Real-Time Collaboration Board

Real-time Collaboration Board with Vue3 + Socket.io

## Project Architecture

- Runtime: NodeJS with Express API + Vite
- Language: TypeScript
- Database: SQLite
- Front-end: Vue3, Vite, Quasar, Pinia
- Back-end: NodeJS, Socket.io
- HTTP Client: Axios

## Setup

### Prerequisites

- Node.js 16+
- QuasarCLI
- SQLite
- npm 6+

### 1. Install the front-end dependencies

inside the project folder run:

```bash
npm install
```

Anywhere in your computer run this command to install the Quasar CLI which is the recommended way to use Quasar.

```bash
npm i -g @quasar/cli
```

### Lint the files

```bash
yarn lint
# or
npm run lint
```

### Format the files

```bash
yarn format
# or
npm run format
```

### Build the app for production

```bash
quasar build
```

## Run

### Start the app in development mode (hot-code reloading, error reporting, etc.)

```bash
quasar dev
```

### Start the server

```bash
npm run server
```
